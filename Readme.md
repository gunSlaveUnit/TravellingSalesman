# Программа "Путешествие"

Данная программа является реализацией решения популярной NP-полной задачи коммивояжёра. 

Решение основано на применении жадного алгоритма, который находит ближайшую точку к текущей точке, пока не обойдет все точки. 

Находит кратчайший замкнутый маршрут с объездом всех требуемых городов.

Информация о маршрутах содержится в файле "travels.bin"

Для построения маршрута:

1. Убедитесь, что файл travels.bin расположен в одной директории с исполняемым файлом.

2. Убедитесь, что в файл travels.bin записаны нужные Вам пути.
(Запустите программу, выберите пункт "Просмотр маршрутов", 
при отсутствии нужных впишите их через пункт "Добавить маршрут").

3. Запустите программу и выберите пункт "Построение маршрута".

4. Введите через запятую список городов для посещения 
(недопускается использование других знаков).

5. Получите ответ.
